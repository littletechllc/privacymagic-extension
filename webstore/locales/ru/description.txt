Веб — замечательная вещь: он даёт доступ к мировой информации. Это неотъемлемая часть современной жизни. К сожалению, веб также является инструментом массовой слежки. Всё, что вы делаете в интернете, собирается, хранится и анализируется корпорациями и правительствами.

Privacy Magic обеспечивает полную многоуровневую защиту от слежки.

Чтобы защитить вашу конфиденциальность, Privacy Magic:

- Блокирует рекламу, трекеры и помехи
- Защищает системную информацию (от снятия отпечатка)
  · Батарея
  · Браузер
  · Устройство
  · Диск
  · Дисплей
  · Шрифты
  · Графика
  · Клавиатура
  · Язык
  · Математика
  · Память
  · Сеть
  · Процессор
  · Экран
  · Часовой пояс
  · Тайминг
  · Сенсор
- Удаляет следы отслеживания
  · Реферер
  · Параметры отслеживания
  · Имя окна
- Укрепляет веб-элементы
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Отправляет сигнал Global Privacy Control
- Отключает функции браузера с утечками
  · Сторонние cookie
  · Наборы связанных сайтов
  · WebRTC UDP без прокси
  · Аудит гиперссылок
- Отключает навязчивые облачные службы
  · Служба проверки орфографии
  · Подсказки поиска
  · Альтернативные страницы ошибок
  · Расширенная отчётность безопасного просмотра
- Отключает API таргетированной рекламы
  · Topics API
  · Protected Audience API
  · Измерение рекламы
