ウェブは素晴らしいものです。世界の情報にアクセスできます。現代生活に欠かせません。残念ながら、ウェブは大規模な監視の手段にもなっています。オンラインでの行動は、企業や政府によって収集、保存、分析されています。

Privacy Magic は、監視からあなたを守るための包括的な多層保護を提供します。

プライバシーを守るため、Privacy Magic は次のことを行います：

- 広告、トラッカー、迷惑コンテンツをブロック
- システム情報を保護（フィンガープリント対策）
  · バッテリー
  · ブラウザ
  · デバイス
  · ディスク
  · 表示
  · フォント
  · グラフィックス
  · キーボード
  · 言語
  · 数学
  · メモリ
  · ネットワーク
  · プロセッサ
  · 画面
  · タイムゾーン
  · タイミング
  · タッチ
- トラッキングの痕跡を削除
  · リファラー
  · トラッキングパラメータ
  · ウィンドウ名
- ウェブ要素を強化
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Global Privacy Control シグナルを送信
- データが漏れるブラウザ機能を無効化
  · サードパーティ Cookie
  · 関連ウェブサイトセット
  · WebRTC 非プロキシ UDP
  · ハイパーリンク監査
- 侵入的なクラウドサービスを無効化
  · スペルチェックサービス
  · 検索候補
  · 代替エラーページ
  · セーフブラウジング拡張レポート
- 広告ターゲティング API を無効化
  · Topics API
  · Protected Audience API
  · 広告測定
