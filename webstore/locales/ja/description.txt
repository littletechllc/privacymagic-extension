この拡張機能は、最大限のプライバシーと使いやすさのために設計された最先端の保護を提供します。すぐ使えます：設定は不要です！

主な保護機能

• 広告とトラッカーをブロック：プライバシーを侵害する広告、追跡スクリプト、ピクセルがブラウザで読み込まれるのを防ぎます。
• 高度なフィンガープリント耐性：あなたを特定しうる 18 以上のシステム／ハードウェア信号（Canvas、WebGL、AudioContext、フォント、タイムゾーンなど）を偽装します。
• 追跡の痕跡を削除：トラッキングパラメータを止め、Referer ヘッダーを最小化し、キャッシュ漏洩を防ぎます。
• ブラウザのプライバシーを強化：サードパーティ Cookie と Topics API をブロックし、Global Privacy Control を適用します。

厳格なプライバシー保証

• 100% オープンソース：コードは誰でも検証できるように公開されています。
• データ収集ゼロ：閲覧活動や個人情報を記録・追跡・販売することはありません（今後もありません）。
• テレメトリゼロ：拡張機能やウェブサイトに Google Analytics などのトラッカーは埋め込まれていません。

Privacy Magic は活発に開発中です。フィードバックやご質問は contact@littletechllc.com までお寄せください。

作者について

Privacy Magic は、ウェブのプライバシーを守るオープンソースソフトウェアに取り組むソフトウェアエンジニア、Arthur Edelstein によって作られています：

• Tor では、Tor Browser にストリーム分離を導入しました。
• Firefox では、Total Cookie Protection と HTTPS-Only Mode を開発したチームのプロダクトマネージャーでした。
• Brave では、HTTPS by Default を導入し、ブラウザのプライバシー脆弱性に関する研究を共著しました。
• ブラウザのプライバシー漏洩を検証するオープンソースの取り組み PrivacyTests.org の創設者です。

技術仕様

- 広告、トラッカー、迷惑コンテンツをブロック
- システム情報を保護（フィンガープリント対策）
  · オーディオ (AudioContext)
  · キーボード
  · グラフィックス (Canvas, WebGL)
  · タイミング
  · タイムゾーン
  · タッチ
  · ディスク
  · デバイス
  · ネットワーク
  · バッテリー
  · フォント
  · ブラウザ (User agent)
  · プロセッサ
  · メモリ
  · 画面
  · 言語
  · 数学
  · 表示
- トラッキングの痕跡を削除
  · ウィンドウ名
  · キャッシュの漏洩
  · トラッキングパラメータ
  · リファラー
- ウェブ要素を強化
  · Iframes
  · Service Workers
  · Shared Storage
  · Shared Workers
  · Web Workers
- Global Privacy Control シグナルを送信
- データが漏れるブラウザ機能を無効化
  · WebRTC 非プロキシ UDP
  · サードパーティ Cookie
  · ハイパーリンク監査
  · 関連ウェブサイトセット
- 侵入的なクラウドサービスを無効化
  · スペルチェックサービス
  · セーフブラウジング拡張レポート
  · 検索候補
  · 代替エラーページ
- 広告ターゲティング API を無効化
  · Protected Audience API
  · Topics API
  · 広告測定 API
- 閲覧履歴の同期をオフにする手助けをします
