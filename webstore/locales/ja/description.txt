ウェブは素晴らしいものです。世界の情報にアクセスできます。現代生活に欠かせません。残念ながら、ウェブは大規模な監視の手段にもなっています。オンラインでの行動は、企業や政府によって収集、保存、分析されています。

Privacy Magic は、監視からあなたを守るための包括的な多層保護を提供します。

プライバシーを守るため、Privacy Magic は次のことを行います：

- 広告、トラッカー、迷惑コンテンツをブロック
- システム情報を保護（フィンガープリント対策）
  · キーボード
  · グラフィックス
  · タイミング
  · タイムゾーン
  · タッチ
  · ディスク
  · デバイス
  · ネットワーク
  · バッテリー
  · フォント
  · ブラウザ
  · プロセッサ
  · メモリ
  · 画面
  · 言語
  · 数学
  · 表示
- トラッキングの痕跡を削除
  · ウィンドウ名
  · トラッキングパラメータ
  · リファラー
- ウェブ要素を強化
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Global Privacy Control シグナルを送信
- データが漏れるブラウザ機能を無効化
  · WebRTC 非プロキシ UDP
  · サードパーティ Cookie
  · ハイパーリンク監査
  · 関連ウェブサイトセット
- 侵入的なクラウドサービスを無効化
  · スペルチェックサービス
  · セーフブラウジング拡張レポート
  · 検索候補
  · 代替エラーページ
- 広告ターゲティング API を無効化
  · Protected Audience API
  · Topics API
  · 広告測定
