Інтернет — чудова річ: він дає вам доступ до світової інформації. Це невід’ємна частина сучасного життя. На жаль, інтернет також є інструментом масового стеження. Усе, що ви робите в мережі, збирається, зберігається та аналізується корпораціями та урядами.

Privacy Magic забезпечує повноцінний багаторівневий захист від стеження.

Щоб захистити вашу конфіденційність, Privacy Magic:

- Блокує рекламу, трекери та зайве
- Захищає системну інформацію (від зняття відбитка)
  · Батарея
  · Браузер
  · Пристрій
  · Диск
  · Дисплей
  · Шрифти
  · Графіка
  · Клавіатура
  · Мова
  · Math
  · Пам'ять
  · Мережа
  · Процесор
  · Екран
  · Часовий пояс
  · Таймінг
  · Сенсор
- Видаляє сліди відстеження
  · Реферер
  · Параметри відстеження
  · Ім'я вікна
- Підсилює веб-елементи
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Надсилає сигнал Global Privacy Control
- Вимикає функції браузера з витоками
  · Сторонні cookie
  · Набори пов’язаних сайтів
  · WebRTC UDP без проксі
  · Аудит гіперпосилань
- Вимикає нав’язливі хмарні служби
  · Служба перевірки орфографії
  · Підказки пошуку
  · Альтернативні сторінки помилок
  · Розширена звітність безпечного перегляду
- Вимикає API таргетингу реклами
  · Topics API
  · Protected Audience API
  · Вимірювання реклами
