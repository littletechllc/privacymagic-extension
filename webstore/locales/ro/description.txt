Web-ul este minunat: vă oferă acces la informațiile lumii. Este o parte esențială a vieții moderne. Din păcate, web-ul este și un vehicul pentru supraveghere în masă. Tot ce faceți online este colectat, stocat și analizat de corporații și guverne.

Privacy Magic oferă protecții temeinice, pe straturi, împotriva supravegherii.

Pentru a vă apăra confidențialitatea, Privacy Magic:

- Blochează reclame, trackere și supărări
- Protejează informațiile de sistem (împotriva fingerprintingului)
  · Afișaj
  · Baterie
  · Browser
  · Cronometrare
  · Disc
  · Dispozitiv
  · Ecran
  · Fonturi
  · Fus orar
  · Grafice
  · Limbă
  · Matematică
  · Memorie
  · Procesor
  · Rețea
  · Tactil
  · Tastatură
- Elimină urmele de urmărire
  · Nume fereastră
  · Parametri urmărire
  · Referrer
- Întărește elementele web
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Trimite semnal Global Privacy Control
- Dezactivează funcțiile browser care scurg date
  · Auditarea hiperlinkurilor
  · Cookie-uri terțe
  · Seturi de site-uri înrudite
  · WebRTC UDP neproxiat
- Dezactivează serviciile cloud invazive
  · Paginile de eroare alternative
  · Raportarea extinsă pentru navigare sigură
  · Serviciul de verificare ortografică
  · Sugestiile de căutare
- Dezactivează API-urile de targetare a reclamelor
  · Măsurarea reclamelor
  · Protected Audience API
  · Topics API
- Vă ajută să dezactivați sincronizarea istoricului
