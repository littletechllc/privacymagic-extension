وب چیز شگفت‌انگیزی است: به شما امکان دسترسی به اطلاعات جهان را می‌دهد. بخش ضروری زندگی مدرن است. متأسفانه وب همچنین وسیله‌ای برای نظارت جمعی است. هر آنچه آنلاین انجام می‌دهید توسط شرکت‌ها و دولت‌ها جمع‌آوری، ذخیره و تجزیه و تحلیل می‌شود.

Privacy Magic محافظت‌های جامع و چندلایه در برابر نظارت ارائه می‌دهد.

برای دفاع از حریم خصوصی شما، Privacy Magic:

- تبلیغات، ردیاب‌ها و مزاحمت‌ها را مسدود می‌کند
- از اطلاعات سیستم محافظت می‌کند (در برابر اثر انگشت)
  · باتری
  · مرورگر
  · دستگاه
  · دیسک
  · نمایش
  · فونت
  · گرافیک
  · صفحه‌کلید
  · زبان
  · Math
  · حافظه
  · شبکه
  · پردازنده
  · صفحه‌نمایش
  · منطقه زمانی
  · زمان‌بندی
  · لمس
- ردپای ردیابی را حذف می‌کند
  · ارجاع‌دهنده
  · پارامترهای ردیابی
  · نام پنجره
- عناصر وب را تقویت می‌کند
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- سیگنال Global Privacy Control ارسال می‌کند
- ویژگی‌های مرورگر نشت‌کننده را غیرفعال می‌کند
  · کوکی‌های شخص ثالث
  · مجموعه‌های سایت مرتبط
  · WebRTC UDP بدون پراکسی
  · ممیزی ابرپیوند
- خدمات ابری تهاجمی را غیرفعال می‌کند
  · سرویس املا
  · پیشنهاد جستجو
  · صفحات خطای جایگزین
  · گزارش‌دهی گسترده مرور امن
- APIهای هدف‌گیری تبلیغات را غیرفعال می‌کند
  · Topics API
  · Protected Audience API
  · اندازه‌گیری تبلیغات
