Web là điều tuyệt vời: nó mang đến cho bạn quyền truy cập thông tin của thế giới. Đó là phần thiết yếu của cuộc sống hiện đại. Thật không may, web cũng là phương tiện giám sát hàng loạt. Mọi thứ bạn làm trực tuyến đều bị thu thập, lưu trữ và phân tích bởi các tập đoàn và chính phủ.

Privacy Magic cung cấp sự bảo vệ toàn diện, nhiều lớp chống lại giám sát.

Để bảo vệ quyền riêng tư của bạn, Privacy Magic:

- Chặn quảng cáo, trình theo dõi và nội dung phiền nhiễu
- Bảo vệ thông tin hệ thống (chống lấy dấu vân tay)
  · Bàn phím
  · Bộ nhớ
  · Bộ xử lý
  · Cảm ứng
  · Đĩa
  · Đồ họa
  · Hiển thị
  · Màn hình
  · Mạng
  · Múi giờ
  · Ngôn ngữ
  · Phông chữ
  · Pin
  · Thiết bị
  · Thời gian
  · Toán
  · Trình duyệt
- Xóa dấu vết theo dõi
  · Người giới thiệu
  · Tên cửa sổ
  · Tham số theo dõi
- Củng cố các phần tử web
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Gửi tín hiệu Global Privacy Control
- Tắt các tính năng trình duyệt rò rỉ
  · Cookie bên thứ ba
  · Kiểm tra siêu liên kết
  · Tập trang web liên quan
  · WebRTC UDP không qua proxy
- Tắt các dịch vụ đám mây xâm phạm
  · Báo cáo mở rộng duyệt web an toàn
  · Dịch vụ chính tả
  · Gợi ý tìm kiếm
  · Trang lỗi thay thế
- Tắt API nhắm mục tiêu quảng cáo
  · Đo lường quảng cáo
  · Protected Audience API
  · Topics API
