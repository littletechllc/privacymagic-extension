Diese Erweiterung bietet modernste Schutzmaßnahmen für maximale Privatsphäre und Nutzerfreundlichkeit. Sofort einsatzbereit: keine Konfiguration nötig!

SCHUTZFUNKTIONEN

• Werbung & Tracker blockieren: Stoppt datenschutzinvasive Werbung, Tracking-Skripte und Pixel, bevor sie in Ihrem Browser geladen werden.
• Erweiterter Fingerprinting-Schutz: Verschleiert 18+ System- und Hardwaresignale (Canvas, WebGL, AudioContext, Schriftarten, Zeitzone und mehr), mit denen Sie sonst identifiziert werden könnten.
• Breadcrumb-Entfernung: Stoppt Tracking-Parameter, minimiert Referrer-Header und verhindert Cache-Leaks.
• Browser-Privatsphäre härten: Blockiert Drittanbieter-Cookies und die Topics API und setzt den Global Privacy Control durch.

STRIKTE DATENSCHUTZGARANTIE

• 100% Open Source: Unser Code ist offen und kann von jedem geprüft und verifiziert werden.
• Keine Datenerfassung: Wir protokollieren, verfolgen oder verkaufen Ihre Surfaktivitäten oder personenbezogenen Daten nicht (und werden das nie tun).
• Keine Telemetrie: Weder in der Erweiterung noch auf der Website ist Google Analytics oder ein anderer Tracker eingebettet.

ÜBER DEN AUTOR

Privacy Magic wurde von Arthur Edelstein entwickelt, einem Softwareingenieur, der Open-Source-Software zum Schutz der Web-Privatsphäre baut:

• Bei Tor führte er Stream Isolation in Tor Browser ein.
• Bei Firefox war er Product Manager der Teams hinter Total Cookie Protection und HTTPS-Only Mode.
• Bei Brave führte er HTTPS by Default ein und war Mitautor von Forschung zu Browser-Privatsphäre-Schwachstellen.
• Gründer von PrivacyTests.org, einer Open-Source-Initiative, die Browser auf Datenschutzlecks testet.

TECHNISCHE SPEZIFIKATIONEN

- Blockiert Werbung, Tracker und Störungen
- Schützt Systeminformationen (gegen Fingerprinting)
  · Anzeige
  · Audio (AudioContext)
  · Batterie
  · Bildschirm
  · Browserversion (User agent)
  · Gerät
  · Grafikchip (Canvas, WebGL)
  · Mathematik
  · Netzwerk
  · Prozessor
  · Schriftarten
  · Speicher
  · Speicherplatz
  · Sprache
  · Tastatur
  · Timer
  · Touch
  · Zeitzone
- Entfernt Tracking-Spuren
  · Cache-Lecks
  · Fenster-Name
  · Referrer-Richtlinie
  · Tracking-Parameter
- Härtet Web-Elemente ab
  · Iframes
  · Service Workers
  · Shared Storage
  · Shared Workers
  · Web Workers
- Sendet Global Privacy Control Signal
- Deaktiviert undichte Browser-Funktionen
  · Drittanbieter-Cookies
  · Hyperlink-Überwachung
  · Verwandte Website-Sets
  · WebRTC UDP ohne Proxy
- Deaktiviert invasive Cloud-Dienste
  · Alternative Fehlerseiten
  · Erweiterte Safe-Browsing-Berichte
  · Rechtschreibprüfung
  · Suchvorschläge
- Deaktiviert API für zielgerichtete Werbung
  · Anzeigenmessung API
  · Protected Audience API
  · Topics API
- Hilft Ihnen, die Verlaufssynchronisierung zu deaktivieren
