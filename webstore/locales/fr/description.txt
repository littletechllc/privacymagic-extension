Le Web est une merveille : il vous donne accès à l'information du monde. C'est une partie essentielle de la vie moderne. Malheureusement, le Web est aussi un vecteur de surveillance de masse. Tout ce que vous faites en ligne est collecté, stocké et analysé par des entreprises et des gouvernements.

Privacy Magic fournit des protections complètes et multicouches pour vous protéger de la surveillance.

Pour défendre votre vie privée, Privacy Magic :

- Bloque les publicités, traceurs et nuisances
- Protège les informations système (pour résister au fingerprinting)
  · Affichage
  · Appareil
  · Batterie
  · Chronométrage
  · Clavier
  · Disque
  · Écran
  · Fuseau horaire
  · Graphique
  · Langue
  · Maths
  · Mémoire
  · Navigateur
  · Polices
  · Processeur
  · Réseau
  · Tactile
- Supprime les traces de suivi
  · Nom de fenêtre
  · Paramètres de suivi
  · Référent
- Durcit les éléments web
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Envoie le signal Global Privacy Control
- Désactive les fonctionnalités du navigateur qui fuient
  · Audit des hyperliens
  · Cookies tiers
  · Ensembles de sites associés
  · WebRTC UDP sans proxy
- Désactive les services cloud invasifs
  · Correcteur orthographique
  · Pages d'erreur alternatives
  · Rapport étendu de navigation sécurisée
  · Suggestions de recherche
- Désactive les API de ciblage publicitaire
  · Mesure publicitaire
  · Protected Audience API
  · Topics API
