Tīmeklis ir brīnišķīga lieta: tas sniedz jums piekļuvi pasaules informācijai. Tas ir būtiska mūsdienu dzīves sastāvdaļa. Diemžēl tīmeklis ir arī masu uzraudzības līdzeklis. Viss, ko darāt tiešsaistē, tiek vākts, glabāts un analizēts korporācijām un valdībām.

Privacy Magic nodrošina rūpīgu, daudzslāņu aizsardzību pret uzraudzību.

Lai aizstāvētu jūsu privātumu, Privacy Magic:

- Bloķē reklāmas, izsekotājus un kaitinājumus
- Aizsargā sistēmas informāciju (pret pirkstu nospiedumu)
  · Atmiņa
  · Baterija
  · Disks
  · Displejs
  · Ekrāns
  · Grafika
  · Ierīce
  · Laika zona
  · Laiks
  · Matemātika
  · Pārlūks
  · Pieskāriens
  · Procesors
  · Šrifti
  · Tastatūra
  · Tīkls
  · Valoda
- Noņem izsekošanas pēdas
  · Atsauce
  · Izsekošanas parametri
  · Loga nosaukums
- Stiprina tīmekļa elementus
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Nosūta Global Privacy Control (GPC) signālu
- Atspējo noplūstošās pārlūka funkcijas
  · Hipersaites audits
  · Saistītās vietņu kopas
  · Trešo pušu sīkdatnes
  · WebRTC bez starpniekservera UDP
- Atspējo iejaukšanās mākoņpakalpojumus
  · Alternatīvās kļūdu lapas
  · Drošās pārlūkošanas paplašinātā ziņošana
  · Meklēšanas ieteikumi
  · Pareizrakstības pakalpojums
- Atspējo reklāmu mērķēšanas API
  · Protected Audience API
  · Reklāmu mērīšana
  · Topics API
