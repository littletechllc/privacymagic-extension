El web és una cosa meravellosa: us dóna accés a la informació del món. És una part essencial de la vida moderna. Malauradament, el web és també un vehicle de vigilància massiva. Tot el que feu en línia es recull, s'emmagatzema i s'analitza per corporacions i governs.

Privacy Magic proporciona proteccions exhaustives i en capes contra la vigilància.

Per defensar la vostra privacitat, Privacy Magic:

- Bloqueja anuncis, seguidors i molesties
- Protegeix la informació del sistema (contra la presa d'empremtes)
  · Bateria
  · Disc
  · Dispositiu
  · Fonts
  · Gràfics
  · Idioma
  · Matemàtiques
  · Memòria
  · Monitor
  · Navegador
  · Pantalla
  · Processador
  · Tàctil
  · Teclat
  · Temps
  · Xarxa
  · Zona horària
- Elimina les restes de seguiment
  · Nom de la finestra
  · Paràmetres de seguiment
  · Referent
- Endureix els elements web
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Envia el senyal Global Privacy Control
- Desactiva les funcions del navegador que perden dades
  · Auditoria d'hiperenllaços
  · Conjunts de llocs relacionats
  · Galetes de tercers
  · WebRTC UDP sense proxy
- Desactiva els serveis al núvol invasius
  · Informe ampliat de navegació segura
  · Pàgines d'error alternatives
  · Servei d'ortografia
  · Suggeriments de cerca
- Desactiva les API de segmentació d'anuncis
  · Mesura d'anuncis
  · Protected Audience API
  · Topics API
