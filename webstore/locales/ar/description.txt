الويب شيء رائع: إنه يمنحك الوصول إلى معلومات العالم. إنه جزء أساسي من الحياة الحديثة. لسوء الحظ، الويب هو أيضًا أداة للمراقبة الجماعية. كل ما تفعله عبر الإنترنت يتم جمعه وتخزينه وتحليله من قبل الشركات والحكومات.

يوفر Privacy Magic حماية شاملة ومتعددة الطبقات ضد المراقبة.

للدفاع عن خصوصيتك، يوفر Privacy Magic:

- حظر الإعلانات والمتتبعات والمضايقات
- حماية معلومات النظام (ضد البصمة الرقمية)
  · البطارية
  · المتصفح
  · الجهاز
  · القرص
  · العرض
  · الخطوط
  · الرسومات
  · لوحة المفاتيح
  · اللغة
  · الرياضيات
  · الذاكرة
  · الشبكة
  · المعالج
  · الشاشة
  · المنطقة الزمنية
  · التوقيت
  · اللمس
- إزالة آثار التتبع
  · المُحيل
  · معلمات التتبع
  · اسم النافذة
- تعزيز عناصر الويب
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- إرسال إشارة Global Privacy Control
- تعطيل ميزات المتصفح التي تتسرب منها البيانات
  · ملفات تعريف الارتباط من جهات خارجية
  · مجموعات المواقع ذات الصلة
  · WebRTC UDP غير المعتمد على وكيل
  · تدقيق الارتباطات التشعبية
- تعطيل خدمات السحابة المتطفلة
  · خدمة التدقيق الإملائي
  · اقتراحات البحث
  · صفحات الخطأ البديلة
  · التقارير الموسعة للتصفح الآمن
- تعطيل واجهات API لاستهداف الإعلانات
  · Topics API
  · Protected Audience API
  · قياس الإعلانات
