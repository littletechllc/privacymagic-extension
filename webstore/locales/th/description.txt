ส่วนขนี้มอบการป้องกันล้ำสมัย ออกแบบเพื่อความเป็นส่วนตัวและการใช้งานสูงสุด ใช้งานได้ทันที: ไม่ต้องตั้งค่า!

การป้องกันหลัก

• บล็อกโฆษณาและตัวติดตาม: หยุดโฆษณาที่ละเมิดความเป็นส่วนตัว สคริปต์ติดตาม และพิกเซลไม่ให้โหลดในเบราว์เซอร์
• ต้านการเก็บลายนิ้วมือขั้นสูง: ปลอมแปลงสัญญาณระบบและฮาร์ดแวร์ 18+ รายการ (Canvas, WebGL, AudioContext, แบบอักษร, เขตเวลา และอื่นๆ) ที่อาจใช้ระบุตัวคุณ
• ลบร่องรอย: หยุดพารามิเตอร์ติดตาม ลดส่วนหัว Referer และป้องกันการรั่วของแคช
• เสริมความเป็นส่วนตัวของเบราว์เซอร์: บล็อกคุกกี้บุคคลที่สามและ Topics API และบังคับใช้ Global Privacy Control

การรับประกันความเป็นส่วนตัวอย่างเข้มงวด

• โอเพนซอร์ส 100%: โค้ดของเราเปิดให้ทุกคนตรวจสอบและยืนยันได้
• ไม่เก็บข้อมูล: เราไม่บันทึก ไม่ติดตาม หรือขาย (และจะไม่ทำ) กิจกรรมการท่องเว็บหรือข้อมูลส่วนบุคคลของคุณ
• ไม่มีเทเลเมทรี: ไม่มี Google Analytics หรือตัวติดตามอื่นในส่วนขหรือเว็บไซต์

เกี่ยวกับผู้เขียน

Privacy Magic สร้างโดย Arthur Edelstein วิศวกรซอฟต์แวร์ที่มุ่งสร้างซอฟต์แวร์โอเพนซอร์สเพื่อปกป้องความเป็นส่วนตัวบนเว็บ:

• ที่ Tor เขาได้นำการแยกสตรีมมาใช้ใน Tor Browser
• ที่ Firefox เขาเป็นผู้จัดการผลิตภัณฑ์ของทีม Total Cookie Protection และ HTTPS-Only Mode
• ที่ Brave เขานำ HTTPS by Default มาใช้และร่วมเขียนงานวิจัยเกี่ยวกับช่องโหว่ความเป็นส่วนตัวของเบราว์เซอร์
• ผู้สร้าง PrivacyTests.org โครงการโอเพนซอร์สที่ทดสอบการรั่วไหลของความเป็นส่วนตัวในเบราว์เซอร์

- ปกป้องข้อมูลระบบ (จากการสร้างลายนิ้วมือ)

  · การสัมผัส
  · เขตเวลา
  · คณิตศาสตร์

  · จอแสดงผล
  · ดิสก์
  · ไทมิง
  · เบราว์เซอร์ (User agent)
  · แบตเตอรี่
  · แป้นพิมพ์
  · โปรเซสเซอร์

  · หน้าจอ
  · อุปกรณ์

  · การรั่วไหลของแคช
  · ชื่อหน้าต่าง
  · ผู้ส่งต่อ
  · พารามิเตอร์ติดตาม

  · Iframes

  · Shared Workers
  · Web Workers
- ส่งสัญญาณ Global Privacy Control

  · หน้าข้อผิดพลาดทางเลือก

  · การวัดโฆษณา API
  · Protected Audience API
  · Topics API
- ช่วยให้คุณปิดการซิงค์ประวัติ
