เว็บเป็นสิ่งมหัศจรรย์: มันให้คุณเข้าถึงข้อมูลของโลก เป็นส่วนสำคัญของชีวิตสมัยใหม่ น่าเสียดายที่เว็บยังเป็นเครื่องมือในการสอดส่องมวลชน ทุกอย่างที่คุณทำออนไลน์ถูกรวบรวม เก็บไว้ และวิเคราะห์โดยบรรษัทและรัฐบาล

Privacy Magic ให้การป้องกันที่ครอบคลุมและเป็นชั้นเพื่อปกป้องคุณจากการสอดส่อง

เพื่อปกป้องความเป็นส่วนตัวของคุณ Privacy Magic:

- บล็อกโฆษณา ตัวติดตาม และสิ่งรบกวน
- ปกป้องข้อมูลระบบ (จากการสร้างลายนิ้วมือ)
  · กราฟิก
  · การสัมผัส
  · เขตเวลา
  · คณิตศาสตร์
  · เครือข่าย
  · จอแสดงผล
  · ดิสก์
  · ไทมิง
  · เบราว์เซอร์
  · แบตเตอรี่
  · แป้นพิมพ์
  · โปรเซสเซอร์
  · ฟอนต์
  · ภาษา
  · หน่วยความจำ
  · หน้าจอ
  · อุปกรณ์
- ลบร่องรอยการติดตาม
  · ชื่อหน้าต่าง
  · ผู้ส่งต่อ
  · พารามิเตอร์ติดตาม
- เสริมความแข็งแกร่งองค์ประกอบเว็บ
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- ส่งสัญญาณ Global Privacy Control
- ปิดฟีเจอร์เบราว์เซอร์ที่รั่ว
  · การตรวจสอบไฮเปอร์ลิงก์
  · คุกกี้ของบุคคลที่สาม
  · ชุดเว็บไซต์ที่เกี่ยวข้อง
  · WebRTC UDP ที่ไม่ผ่านพร็อกซี
- ปิดบริการคลาวด์ที่รุกราน
  · การรายงานขยายการท่องเว็บอย่างปลอดภัย
  · คำแนะนำการค้นหา
  · บริการตรวจการสะกด
  · หน้าข้อผิดพลาดทางเลือก
- ปิด API การกำหนดเป้าหมายโฆษณา
  · การวัดโฆษณา
  · Protected Audience API
  · Topics API
