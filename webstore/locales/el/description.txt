Αυτή η επέκταση παρέχει σύγχρονες προστασίες, σχεδιασμένες για μέγιστη ιδιωτικότητα και χρηστικότητα. Λειτουργεί αμέσως: δεν απαιτείται ρύθμιση!

ΒΑΣΙΚΕΣ ΠΡΟΣΤΑΣΙΕΣ

• Αποκλεισμός διαφημίσεων και trackers: Σταματά τη φόρτωση επεμβατικών διαφημίσεων, σεναρίων παρακολούθησης και pixel στο πρόγραμμα περιήγησης.
• Προηγμένη αντοχή στο fingerprinting: Παραποιεί 18+ σήματα συστήματος και υλικού (Canvas, WebGL, AudioContext, γραμματοσειρές, ζώνη ώρας κ.ά.) που αλλιώς θα μπορούσαν να σας ταυτοποιήσουν.
• Αφαίρεση ιχνών: Σταματά παραμέτρους παρακολούθησης, ελαχιστοποιεί κεφαλίδες Referer και αποτρέπει διαρροές cache.
• Ενίσχυση ιδιωτικότητας προγράμματος περιήγησης: Αποκλείει cookies τρίτων και το Topics API και εφαρμόζει το Global Privacy Control.

ΑΥΣΤΗΡΗ ΕΓΓΥΗΣΗ ΙΔΙΩΤΙΚΟΤΗΤΑΣ

• 100% ανοιχτός κώδικας: Ο κώδικάς μας είναι ανοιχτός για εξέταση και επαλήθευση από όλους.
• Μηδενική συλλογή δεδομένων: Δεν καταγράφουμε, παρακολουθούμε ή πουλάμε (και δεν θα το κάνουμε ποτέ) τη δραστηριότητα περιήγησής σας ή προσωπικά δεδομένα.
• Μηδενική τηλεμετρία: Κανένα Google Analytics ή άλλος tracker δεν είναι ενσωματωμένος στην επέκταση ή στον ιστότοπο.

Το Privacy Magic είναι σε ενεργή ανάπτυξη. Σχόλια ή ερωτήσεις; Γράψτε μας στο contact@littletechllc.com.

ΣΧΕΤΙΚΑ ΜΕ ΤΟΝ ΣΥΓΓΡΑΦΕΑ

Το Privacy Magic δημιουργείται από τον Arthur Edelstein, μηχανικό λογισμικού αφοσιωμένο σε λογισμικό ανοιχτού κώδικα που υπερασπίζεται την ιδιωτικότητα στο web:

• Στο Tor εισήγαγε την απομόνωση ροών στο Tor Browser.
• Στο Firefox ήταν product manager των ομάδων Total Cookie Protection και HTTPS-Only Mode.
• Στο Brave εισήγαγε το HTTPS by Default και συνυπέγραψε έρευνα για ευπάθειες ιδιωτικότητας προγραμμάτων περιήγησης.
• Δημιουργός του PrivacyTests.org, πρωτοβουλίας ανοιχτού κώδικα που δοκιμάζει διαρροές ιδιωτικότητας προγραμμάτων περιήγησης.

ΤΕΧΝΙΚΕΣ ΠΡΟΔΙΑΓΡΑΦΕΣ

- Αποκλείει διαφημίσεις, παρακολουθητές και ενοχλήσεις
- Προστατεύει πληροφορίες συστήματος (από fingerprinting)
  · Αφή
  · Γλώσσα
  · Γραμματοσειρές
  · Γραφικά (Canvas, WebGL)
  · Δίκτυο
  · Δίσκος
  · Επεξεργαστής
  · Ζώνη ώρας
  · Ήχος (AudioContext)
  · Μαθηματικά
  · Μνήμη
  · Μπαταρία
  · Οθόνη
  · Πληκτρολόγιο
  · Πρόγραμμα περιήγησης (User agent)
  · Ρυθμίσεις οθόνης
  · Συσκευή
  · Χρονομέτρηση
- Αφαιρεί ίχνη παρακολούθησης
  · Αναφορά
  · Διαρροές cache
  · Όνομα παραθύρου
  · Παράμετροι παρακολούθησης
- Ενισχύει στοιχεία web
  · Iframes
  · Service Workers
  · Shared Storage
  · Shared Workers
  · Web Workers
- Αποστέλλει σήμα GPC (Global Privacy Control)
- Απενεργοποιεί λειτουργίες προγράμματος περιήγησης που διαρρέουν
  · Έλεγχος υπερσυνδέσμων
  · Σχετικά σύνολα ιστοτόπων
  · Cookies τρίτων
  · WebRTC UDP χωρίς proxy
- Απενεργοποιεί επεμβατικές υπηρεσίες cloud
  · Εκτεταμένη αναφορά ασφαλούς περιήγησης
  · Εναλλακτικές σελίδες σφάλματος
  · Προτάσεις αναζήτησης
  · Υπηρεσία ορθογραφίας
- Απενεργοποιεί API στόχευσης διαφημίσεων
  · Μέτρηση διαφημίσεων API
  · Protected Audience API
  · Topics API
- Σας βοηθά να απενεργοποιήσετε τον συγχρονισμό ιστορικού
