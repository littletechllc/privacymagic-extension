Το διαδίκτυο είναι κάτι υπέροχο: σας δίνει πρόσβαση στις πληροφορίες του κόσμου. Είναι απαραίτητο μέρος της σύγχρονης ζωής. Δυστυχώς, το διαδίκτυο είναι και μέσο μαζικής παρακολούθησης. Όλα όσα κάνετε online συλλέγονται, αποθηκεύονται και αναλύονται από εταιρείες και κυβερνήσεις.

Το Privacy Magic παρέχει ενδελεχείς, στρωματοποιημένες προστασίες από την παρακολούθηση.

Για να προστατέψει το απόρρητό σας, το Privacy Magic:

- Αποκλείει διαφημίσεις, παρακολουθητές και ενοχλήσεις
- Προστατεύει πληροφορίες συστήματος (από fingerprinting)
  · Μπαταρία
  · Πρόγραμμα περιήγησης
  · Συσκευή
  · Δίσκος
  · Οθόνη
  · Γραμματοσειρές
  · Γραφικά
  · Πληκτρολόγιο
  · Γλώσσα
  · Math
  · Μνήμη
  · Δίκτυο
  · Επεξεργαστής
  · Οθόνη
  · Ζώνη ώρας
  · Χρονομέτρηση
  · Αφή
- Αφαιρεί ίχνη παρακολούθησης
  · Αναφορά
  · Παράμετροι παρακολούθησης
  · Όνομα παραθύρου
- Ενισχύει στοιχεία web
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Αποστέλλει σήμα GPC (Global Privacy Control)
- Απενεργοποιεί λειτουργίες προγράμματος περιήγησης που διαρρέουν
  · Cookies τρίτων
  · Σχετικά σύνολα ιστοτόπων
  · WebRTC UDP χωρίς proxy
  · Έλεγχος υπερσυνδέσμων
- Απενεργοποιεί επεμβατικές υπηρεσίες cloud
  · Υπηρεσία ορθογραφίας
  · Προτάσεις αναζήτησης
  · Εναλλακτικές σελίδες σφάλματος
  · Εκτεταμένη αναφορά ασφαλούς περιήγησης
- Απενεργοποιεί API στόχευσης διαφημίσεων
  · Topics API
  · Protected Audience API
  · Μέτρηση διαφημίσεων
