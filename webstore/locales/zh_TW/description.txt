網路為我們帶來了便利：它讓您能夠取得世界各地的資訊，已成為現代生活不可或缺的一部分。遺憾的是，網路也是大規模監控的管道。您在網上的一舉一動都在被企業與政府收集、儲存與分析。

Privacy Magic 提供全面、多層次的保護，助您抵禦監控。

為守護您的隱私，Privacy Magic：

- 封鎖廣告、追蹤器與騷擾內容
- 保護系統資訊（防止指紋辨識）
  · 字型
  · 計時
  · 時區
  · 記憶體
  · 處理器
  · 裝置
  · 電池
  · 圖形
  · 磁碟
  · 網路
  · 語言
  · 數學
  · 螢幕
  · 鍵盤
  · 瀏覽器
  · 觸控
  · 顯示
- 移除追蹤痕跡
  · 追蹤參數
  · 參照來源
  · 視窗名稱
- 加固網頁元素
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- 發送 Global Privacy Control 訊號
- 關閉易洩漏資料的瀏覽器功能
  · 相關網站集
  · 第三方 Cookie
  · 超連結稽核
  · WebRTC 非代理 UDP
- 關閉侵入式雲端服務
  · 安全瀏覽擴充報告
  · 拼字檢查服務
  · 替代錯誤頁面
  · 搜尋建議
- 關閉廣告定向 API
  · 廣告衡量
  · Protected Audience API
  · Topics API
