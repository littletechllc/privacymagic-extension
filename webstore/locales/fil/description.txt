Ang web ay kahanga-hanga: binibigyan ka nito ng access sa impormasyon ng mundo. Mahalagang bahagi ito ng modernong buhay. Sa kasamaang-palad, ang web ay daluyan din ng mass surveillance. Lahat ng ginagawa mo online ay kinokolekta, iniimbak at sinusuri ng mga korporasyon at gobyerno.

Nagbibigay ang Privacy Magic ng masinsinan, patong-patong na proteksyon laban sa surveillance.

Para ipagtanggol ang iyong privacy, ang Privacy Magic:

- Nagba-block ng ads, trackers at mga abala
- Pinoprotektahan ang system information (laban sa fingerprinting)

  · Browser
  · Device
  · Disk
  · Display
  · Fonts
  · Graphics
  · Keyboard
  · Language
  · Matematika
  · Memory
  · Network
  · Processor
  · Screen

  · Timing

- Inaalis ang tracking breadcrumbs

- Pinapatay ang mga browser feature na tumatagas

- Pinapatay ang mga invasive cloud services
