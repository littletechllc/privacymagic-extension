웹은 훌륭한 것입니다. 전 세계 정보에 접근할 수 있게 해 줍니다. 현대 생활에 필수적입니다. 안타깝게도 웹은 대량 감시의 수단이기도 합니다. 온라인에서 하는 모든 행동이 기업과 정부에 의해 수집, 저장, 분석됩니다.

Privacy Magic은 감시로부터 당신을 보호하는 철저하고 다층적인 보호를 제공합니다.

당신의 프라이버시를 지키기 위해 Privacy Magic은 다음을 제공합니다:

- 광고, 추적기 및 성가신 콘텐츠 차단
- 시스템 정보 보호 (핑거프린팅 방지)
  · 배터리
  · 브라우저
  · 기기
  · 디스크
  · 디스플레이
  · 글꼴
  · 그래픽
  · 키보드
  · 언어
  · 수학
  · 메모리
  · 네트워크
  · 프로세서
  · 화면
  · 시간대
  · 타이밍
  · 터치
- 추적 흔적 제거
  · 리퍼러
  · 추적 매개변수
  · 창 이름
- 웹 요소 강화
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Global Privacy Control 신호 전송
- 데이터 유출 브라우저 기능 비활성화
  · 서드 파티 쿠키
  · 관련 웹사이트 세트
  · WebRTC 비프록시 UDP
  · 하이퍼링크 감사
- 침습적 클라우드 서비스 비활성화
  · 맞춤법 검사 서비스
  · 검색 제안
  · 대체 오류 페이지
  · 안전한 검색 확장 보고
- 광고 타겟팅 API 비활성화
  · Topics API
  · Protected Audience API
  · 광고 측정
