Žiniatinklis yra nuostabus: jis suteikia jums prieigą prie pasaulio informacijos. Tai esminė šiuolaikinio gyvenimo dalis. Deja, žiniatinklis taip pat yra masinės priežiūros priemonė. Viskas, ką darote internete, renka, saugo ir analizuoja korporacijos ir vyriausybės.

Privacy Magic suteikia nuodugnų, sluoksniuotą apsaugą nuo priežiūros.

Jūsų privatumui ginti Privacy Magic:

- Blokuoja reklamas, sekiklius ir erzinimus
- Apsaugo sistemos informaciją (nuo pirštų atspaudų)
  · Atmintis
  · Baterija
  · Diskas
  · Ekranas
  · Ekrano nustatymai
  · Grafika
  · Įrenginys
  · Kalba
  · Klaviatūra
  · Laikmatis
  · Laiko juosta
  · Lietimas
  · Matematika
  · Naršyklė
  · Procesorius
  · Šriftai
  · Tinklas
- Pašalina sekimo pėdsakus
  · Lango pavadinimas
  · Nurodytojas
  · Sekimo parametrai
- Stiprina žiniatinklio elementus
  · Iframes
  · Service Workers
  · Shared Storage
  · Web Workers
- Siunčia Global Privacy Control (GPC) signalą
- Išjungia nesandarius naršyklės funkcijas
  · Hipernuorodų auditas
  · Susijusių svetainių rinkiniai
  · Trečiųjų šalių slapukai
  · WebRTC be proxy UDP
- Išjungia įsibroviančias debesų paslaugas
  · Alternatyvūs klaidų puslapiai
  · Paieškos pasiūlymai
  · Rašybos tikrinimo paslauga
  · Saugaus naršymo išplėstinė ataskaita
- Išjungia reklamų taikinimo API
  · Protected Audience API
  · Reklamų matavimas
  · Topics API
