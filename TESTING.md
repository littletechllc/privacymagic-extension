# TESTING

To test the Privacy Magic browser extension, you need to install it in your browser. Here's how to do that:

1. Go to https://github.com/littletechllc/privacymagic-extension/releases and download a zip file (not the source code) to your Downloads folder.
2. Open Chrome, click on `â‹® > Extensions > Manage Extensions` to open the Extensions page.
3. Click on the `Developer mode` toggle so that Developer Mode is enabled.
4. Open the Downloads folder, and drag the zip file you downloaded and drop it on the Extensions page.
5. Wait a minute: a Privacy Magic card will appear on the Extensions page.
6. On the toolbar at the top of the window, find the Extensions icon
7. Click on the Extensions icon and "pin" the Privacy Magic icon to the toolbar.
8. Now browse as normally! Your privacy should be protected. If you find a website isn't working properly, please report it and we will look for a fix!
